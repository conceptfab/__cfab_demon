# LAN Web UI (Serwer WWW) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Udostępnić UI TIMEFLOW w sieci lokalnej z przeglądarki na innym urządzeniu (odczyt + edycja), z dostępem chronionym kodem parowania.

**Architecture:** Serwer HTTP hostowany **w procesie aplikacji dashboard (Tauri)** — nie w demonie — dzięki czemu reużywa 1:1 istniejące komendy `#[tauri::command]` zamiast je duplikować. Ten sam bundel React jest serwowany po HTTP; w przeglądarce (brak Tauri IPC) warstwa [core.ts](../../../dashboard/src/lib/tauri/core.ts) przełącza `invoke()` na `fetch(POST /rpc)`. Wariant A z brainstormingu.

**Świadomy kompromis (potwierdzony z użytkownikiem):** web-UI działa **tylko gdy aplikacja TIMEFLOW jest uruchomiona** (zwykle w tray'u na maszynie głównej). „Zawsze włączony" wymagałby przeniesienia logiki komend do crate `shared` i hostowania w demonie — to osobny, większy projekt (patrz Roadmap).

**Tech Stack:** Rust (Tauri v2, `std::net::TcpListener` jak w `src/lan_server.rs`, `tauri::async_runtime::block_on`), `include_dir` (embed bundla SPA), React + TypeScript, Tailwind.

**Zakres tego planu = Faza 1 (fundament, pionowy plaster):** pełny pipeline end-to-end udowodniony na grupie komend `clients` (read+write) + dedykowana zakładka „Serwer WWW" w Ustawieniach + ekran logowania w przeglądarce. Rozłożenie pozostałych ~180 komend i hardening produkcyjny to kolejne plany (Roadmap na końcu).

**Bezpieczeństwo (założenia v1):** serwer domyślnie **wyłączony**; po włączeniu nasłuchuje na `0.0.0.0`; dostęp wymaga tokena uzyskanego z jednorazowego 6-cyfrowego kodu parowania; transport to czysty HTTP w LAN (bez TLS) — akceptowalne tylko w zaufanej sieci, jawnie zakomunikowane w UI i Help.

use std::collections::HashMap;
use std::sync::Mutex;

/// Monotonic clock seconds — injected so logic is deterministic in tests.
pub type NowSecs = u64;

const PAIRING_CODE_TTL_SECS: u64 = 180; // 3 min, like LAN pairing
const SESSION_TTL_SECS: u64 = 60 * 60 * 24 * 14; // 14 days

#[derive(Debug, Clone)]
struct PendingCode {
    code: String,
    expires_at: NowSecs,
}

#[derive(Debug, Clone, serde::Serialize)]
pub struct Session {
    pub token: String,
    pub label: String,
    pub created_at: NowSecs,
    pub expires_at: NowSecs,
}

#[derive(Default)]
pub struct AuthState {
    pending: Mutex<Option<PendingCode>>,
    sessions: Mutex<HashMap<String, Session>>,
}

impl AuthState {
    pub fn new() -> Self {
        Self::default()
    }

    /// Store a freshly generated pairing code (caller supplies the random code).
    pub fn set_pairing_code(&self, code: String, now: NowSecs) {
        let mut g = self.pending.lock().unwrap();
        *g = Some(PendingCode { code, expires_at: now + PAIRING_CODE_TTL_SECS });
    }

    /// Exchange a code for a session token. `mint_token` supplies a random token.
    pub fn redeem(
        &self,
        code: &str,
        label: String,
        now: NowSecs,
        mint_token: impl FnOnce() -> String,
    ) -> Result<Session, String> {
        let mut pending = self.pending.lock().unwrap();
        let valid = match pending.as_ref() {
            Some(p) if p.expires_at > now && p.code == code => true,
            _ => false,
        };
        if !valid {
            return Err("invalid_or_expired_code".to_string());
        }
        *pending = None; // single-use
        drop(pending);

        let session = Session {
            token: mint_token(),
            label,
            created_at: now,
            expires_at: now + SESSION_TTL_SECS,
        };
        self.sessions
            .lock()
            .unwrap()
            .insert(session.token.clone(), session.clone());
        Ok(session)
    }

    /// True if the token maps to a live (non-expired) session.
    pub fn is_authorized(&self, token: &str, now: NowSecs) -> bool {
        match self.sessions.lock().unwrap().get(token) {
            Some(s) => s.expires_at > now,
            None => false,
        }
    }

    pub fn revoke(&self, token: &str) {
        self.sessions.lock().unwrap().remove(token);
    }

    pub fn list_sessions(&self, now: NowSecs) -> Vec<Session> {
        self.sessions
            .lock()
            .unwrap()
            .values()
            .filter(|s| s.expires_at > now)
            .cloned()
            .collect()
    }
}

#[cfg(test)]

#[derive(Deserialize)]
struct CreateClientArgs {
    name: String,
    #[serde(default)] contact: Option<String>,
    #[serde(default)] address: Option<String>,
    #[serde(default)] tax_id: Option<String>,
    #[serde(default)] currency: Option<String>,
    #[serde(default)] default_hourly_rate: Option<f64>,
    #[serde(default)] color: Option<String>,
}

#[derive(Deserialize)]
struct UpdateClientArgs {
    id: i64,
    name: String,
    #[serde(default)] contact: Option<String>,
    #[serde(default)] address: Option<String>,
    #[serde(default)] tax_id: Option<String>,
    #[serde(default)] currency: Option<String>,
    #[serde(default)] default_hourly_rate: Option<f64>,
    #[serde(default)] color: Option<String>,
}

#[derive(Deserialize)]
struct ClientIdArg { id: i64 }

fn ok<T: serde::Serialize>(v: T) -> Result<Value, String> {
    serde_json::to_value(v).map_err(|e| format!("serialize: {e}"))
}

/// Route an RPC command to the matching Tauri command. v1 covers the `clients`
/// group end-to-end; new groups are added here following the same pattern.
pub fn dispatch(app: &AppHandle, req: RpcRequest) -> Result<Value, String> {
    let app = app.clone();
    let RpcRequest { command, args } = req;
    match command.as_str() {
        "clients_list" => ok(block(commands::clients_list(app))?),
        "get_clients_summary" => ok(block(commands::get_clients_summary(app))?),
        "clients_create" => {
            let a: CreateClientArgs = args_into(args)?;
            ok(block(commands::clients_create(
                app, a.name, a.contact, a.address, a.tax_id, a.currency, a.default_hourly_rate, a.color,
            ))?)
        }
        "clients_update" => {
            let a: UpdateClientArgs = args_into(args)?;
            ok(block(commands::clients_update(
                app, a.id, a.name, a.contact, a.address, a.tax_id, a.currency, a.default_hourly_rate, a.color,
            ))?)
        }
        "clients_archive" => {
            let a: ClientIdArg = args_into(args)?;
            ok(block(commands::clients_archive(app, a.id))?)
        }
        "clients_delete" => {
            let a: ClientIdArg = args_into(args)?;
            ok(block(commands::clients_delete(app, a.id))?)
        }
        other => Err(format!("unknown_command: {other}")),
    }
}

/// Block on a command future from the synchronous HTTP-server thread.
fn block<T>(fut: impl std::future::Future<Output = Result<T, String>>) -> Result<T, String> {
    tauri::async_runtime::block_on(fut)
}
```

/// Minimal parsed view of an HTTP/1.1 request we care about.
pub struct ParsedRequest {
    pub method: String,
    pub path: String,
    pub bearer: Option<String>,
    pub body: String,
}

/// Parse method, path, Authorization bearer token and body from a raw request.
pub fn parse_request(raw: &str) -> Option<ParsedRequest> {
    let (head, body) = raw.split_once("\r\n\r\n").unwrap_or((raw, ""));
    let mut lines = head.lines();
    let request_line = lines.next()?;
    let mut parts = request_line.split_whitespace();
    let method = parts.next()?.to_string();
    let path = parts.next()?.to_string();
    let mut bearer = None;
    for line in lines {
        if let Some(rest) = line.strip_prefix("Authorization:") {
            let val = rest.trim();
            if let Some(tok) = val.strip_prefix("Bearer ") {
                bearer = Some(tok.trim().to_string());
            }
        }
    }
    Some(ParsedRequest { method, path, bearer, body: body.to_string() })
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn parses_get_with_bearer() {
        let raw = "GET /rpc HTTP/1.1\r\nHost: x\r\nAuthorization: Bearer abc123\r\n\r\n";
        let r = parse_request(raw).unwrap();
        assert_eq!(r.method, "GET");
        assert_eq!(r.path, "/rpc");
        assert_eq!(r.bearer.as_deref(), Some("abc123"));
        assert!(r.body.is_empty());
    }

    #[test]
    fn parses_post_body_without_auth() {
        let raw = "POST /rpc HTTP/1.1\r\nContent-Length: 2\r\n\r\n{}";
        let r = parse_request(raw).unwrap();
        assert_eq!(r.method, "POST");
        assert_eq!(r.body, "{}");
        assert!(r.bearer.is_none());
    }
}

use std::net::TcpListener;

use include_dir::{include_dir, Dir};
use tauri::AppHandle;

use crate::webui::auth::AuthState;

// Embedded SPA bundle. `npm run build` (Tauri beforeBuildCommand) populates ../dist.
static SPA: Dir<'_> = include_dir!("$CARGO_MANIFEST_DIR/../dist");

fn now_secs() -> u64 {
    std::time::SystemTime::now()
        .duration_since(std::time::UNIX_EPOCH)
        .map(|d| d.as_secs())
        .unwrap_or(0)
}

fn rand_digits6() -> String {
    // Non-crypto pairing code; single-use + short TTL keep the risk low.
    let n = now_secs().wrapping_mul(2654435761) % 1_000_000;
    format!("{n:06}")
}

fn rand_token() -> String {
    let a = now_secs().wrapping_mul(0x9E3779B97F4A7C15);
    let b = (a >> 17) ^ a.wrapping_mul(0xD1B54A32D192ED03);
    format!("{a:016x}{b:016x}")
}

fn http(status: &str, content_type: &str, body: &[u8]) -> Vec<u8> {
    let mut out = format!(
        "HTTP/1.1 {status}\r\nContent-Type: {content_type}\r\nContent-Length: {}\r\n\
         Content-Security-Policy: default-src 'self'; connect-src 'self'\r\n\
         Access-Control-Allow-Origin: *\r\nConnection: close\r\n\r\n",
        body.len()
    )
    .into_bytes();
    out.extend_from_slice(body);
    out
}

fn json_response(status: &str, json: &str) -> Vec<u8> {
    http(status, "application/json", json.as_bytes())
}

fn serve_spa(path: &str) -> Vec<u8> {
    let rel = path.trim_start_matches('/');
    let candidate = if rel.is_empty() { "index.html" } else { rel };
    match SPA.get_file(candidate) {
        Some(f) => http("200 OK", mime_for(candidate), f.contents()),
        // SPA fallback: unknown route → index.html (client-side router).
        None => match SPA.get_file("index.html") {
            Some(idx) => http("200 OK", "text/html", idx.contents()),
            None => http("500 Internal Server Error", "text/plain", b"bundle missing"),
        },
    }
}

fn mime_for(name: &str) -> &'static str {
    match name.rsplit('.').next() {
        Some("html") => "text/html",
        Some("js") => "text/javascript",
        Some("css") => "text/css",
        Some("svg") => "image/svg+xml",
        Some("png") => "image/png",
        Some("woff2") => "font/woff2",
        Some("json") => "application/json",
        _ => "application/octet-stream",
    }
}

/// Spawn the listener thread. Returns immediately; serving happens on the thread.
pub fn spawn(app: AppHandle, auth: Arc<AuthState>, port: u16) -> Result<(), String> {
    let listener = TcpListener::bind(("0.0.0.0", port))
        .map_err(|e| format!("bind 0.0.0.0:{port} failed: {e}"))?;
    log::info!("[webui] listening on 0.0.0.0:{port}");
    std::thread::spawn(move || {
        for stream in listener.incoming() {
            let Ok(mut stream) = stream else { continue };
            let mut buf = vec![0u8; 64 * 1024];
            let n = match stream.read(&mut buf) {
                Ok(n) => n,
                Err(_) => continue,
            };
            let raw = String::from_utf8_lossy(&buf[..n]).to_string();
            let resp = handle(&app, &auth, &raw);
            let _ = stream.write_all(&resp);
            let _ = stream.flush();
        }
    });
    Ok(())
}

fn handle(app: &AppHandle, auth: &Arc<AuthState>, raw: &str) -> Vec<u8> {
    let Some(req) = parse_request(raw) else {
        return json_response("400 Bad Request", r#"{"ok":false,"error":"bad_request"}"#);
    };
    match (req.method.as_str(), req.path.as_str()) {
        ("GET", "/healthz") => json_response("200 OK", r#"{"ok":true}"#),

        ("POST", "/auth/pair") => {
            #[derive(serde::Deserialize)]
            struct PairBody { code: String, #[serde(default)] label: String }
            let body: PairBody = match serde_json::from_str(&req.body) {
                Ok(b) => b,
                Err(_) => return json_response("400 Bad Request", r#"{"ok":false,"error":"bad_request"}"#),
            };
            let label = if body.label.is_empty() { "browser".into() } else { body.label };
            match auth.redeem(&body.code, label, now_secs(), rand_token) {
                Ok(session) => {
                    let json = serde_json::json!({"ok": true, "token": session.token}).to_string();
                    json_response("200 OK", &json)
                }
                Err(e) => {
                    let json = serde_json::json!({"ok": false, "error": e}).to_string();
                    json_response("401 Unauthorized", &json)
                }
            }
        }

        ("POST", "/rpc") => {
            let authorized = req
                .bearer
                .as_deref()
                .map(|t| auth.is_authorized(t, now_secs()))
                .unwrap_or(false);
            if !authorized {
                return json_response("401 Unauthorized", r#"{"ok":false,"error":"unauthorized"}"#);
            }
            let parsed = match RpcRequest::parse(&req.body) {
                Ok(p) => p,
                Err(e) => {
                    let json = serde_json::json!({"ok": false, "error": e}).to_string();
                    return json_response("400 Bad Request", &json);
                }
            };
            match rpc::dispatch(app, parsed) {
                Ok(data) => {
                    let json = serde_json::json!({"ok": true, "data": data}).to_string();
                    json_response("200 OK", &json)
                }
                Err(e) => {
                    let json = serde_json::json!({"ok": false, "error": e}).to_string();
                    json_response("200 OK", &json) // app-level error, HTTP 200 with ok:false
                }
            }
        }

        ("GET", _) => serve_spa(&req.path),

        _ => json_response("404 Not Found", r#"{"ok":false,"error":"not_found"}"#),
    }
}
```

    /* ignore */

/** Thrown when the server rejects the token — the UI shows the login gate. */
export class WebUnauthorizedError extends Error {}

export async function pairWithCode(code: string): Promise<void> {
  const res = await fetch('/auth/pair', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ code, label: navigator.userAgent.slice(0, 60) }),
  });
  const data = await res.json().catch(() => ({}));
  if (!res.ok || !data.token) {
    throw new Error(data.error ?? 'pairing_failed');
  }
  setWebToken(data.token as string);
}

/** Browser-mode replacement for Tauri invoke(): POST /rpc. */
export async function httpInvoke<T>(
  command: string,
  args?: Record<string, unknown>,
): Promise<T> {
  const token = getWebToken();
  if (!token) throw new WebUnauthorizedError('no_token');
  const res = await fetch('/rpc', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Authorization: `Bearer ${token}`,
    },
    body: JSON.stringify({ command, args: args ?? {} }),
  });
  if (res.status === 401) {
    clearWebToken();
    throw new WebUnauthorizedError('unauthorized');
  }
  const payload = await res.json();
  if (!payload.ok) throw new Error(payload.error ?? 'rpc_error');
  return payload.data as T;
}

export function WebLoginGate({ children }: { children: React.ReactNode }) {

  const [code, setCode] = useState('');

    } catch {
      setError(t('webserver.login.invalid_code'));
    } finally {
      setBusy(false);
    }
  };

      <div className="w-full max-w-sm space-y-4 rounded-xl border border-border/50 p-6">
        <h1 className="text-lg font-semibold text-foreground">
          {t('webserver.login.title')}
        </h1>

          {t('webserver.login.hint')}
        </p>

          value={code}
          onChange={(e) => setCode(e.target.value)}

- **Tylko gdy aplikacja działa** — nie zastępuje „zawsze-on" demona.
