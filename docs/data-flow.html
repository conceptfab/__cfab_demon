Skrót architektury

Daemon to lekki binar w tle (Rust). Co ~10 s próbkuje aktywne okno i CPU, agreguje dane w pamięci, a co ~5 min zapisuje snapshot dnia do współdzielonej bazy SQLite (daily_store). Dashboard to osobny proces Tauri: React UI + Rust backend ze swoją bazą (projekty, sesje manualne, PM). Synchronizacja w LAN idzie przez wbudowany HTTP serwer daemona na porcie 47891, a online sync przez SFTP + serwer koordynacyjny.
